# 🚀 Ricardo Zermeño -- Developer Portfolio

A modern, responsive, and minimal developer portfolio built with
**React + Vite**, showcasing my work across **data engineering**,
**automation**, and **web development**.

This project serves both as my personal website and as a living
portfolio where I document real-world projects I've built using Python,
SQL, Odoo, and modern frontend technologies.

## 🧰 **Tech Stack**

**Frontend** - React (Vite) - Modern CSS (no frameworks) - Responsive
mobile-first layout - Custom components and animations

**Features** - Mobile hamburger menu + smooth scroll navigation -
Project filtering (All / Data / Web Dev) - Dynamic project cards with
hover animation - Category badges (Data / Web Dev) - Sticky navbar with
blur background - Custom favicon (SVG) - Component architecture (clean
separation)

## 🎨 **UI Features**

### ✅ Project filtering

The portfolio section includes category filters:

-   **All**
-   **Data**
-   **Web Dev**

These instantly update the displayed cards without reloading the page.

### ✅ Card hover animation

Cards elevate smoothly on hover:

-   Subtle transform
-   Soft shadow
-   Maintains accessibility and readability

### ✅ Modern badge system

Each project displays a category badge above the title:

-   Blue = Data\
-   Green = Web Dev

### ✅ Mobile-first navigation

Includes:

-   Sticky navbar\
-   Mobile dropdown menu\
-   Smooth-scroll section navigation\
-   Clean clickable logo returning to "/"

### ✅ Custom Branding

-   `{RZ}` signature icon\
-   Custom SVG favicon\
-   Typography designed for a "developer aesthetic"

## ⚙️ **Installation & Running Locally**

Clone the repo:

``` bash
git clone https://github.com/YOUR-USER/ricardo-portfolio.git
cd ricardo-portfolio
```

Install dependencies:

``` bash
npm install
```

Run dev server:

``` bash
npm run dev
```

Build for production:

``` bash
npm run build
```

Preview built site:

``` bash
npm run preview
```
